# Folklore Generator
Yeoman generator to start projects at Folklore

## Install
```
npm install -g generator-folklore
```

## Usage
```
yo folklore
```

## Sub-Generators

```
yo folklore:laravel
```
Laravel 5 project, JS, Sass, and miscellaneous Laravel boilerplate.

--

```
yo folklore:html
```
HTML static application

--

```
yo folklore:npm-package
```
NPM package
